## Как использовать

1. **Подключение в шаблоне:**

```php
<?php include($_SERVER["DOCUMENT_ROOT"] . "/local/include/cookie_panel.php"); ?>
```

2. **Условное подключение аналитики:**

```php
<?php if (CookieConsent::isAnalyticalAllowed()): ?>
    <!-- Код Яндекс.Метрики -->
<?php endif; ?>
```

3. **Или через метод renderAnalytics:**

```php
<?= CookieConsent::renderAnalytics('<script>/* код аналитики */</script>') ?>
```

## Тестирование

Откройте страницу `/test/cookie-test.php` для проверки работы системы:

- Показывает текущий статус всех типов cookie
- Позволяет очистить cookie для тестирования
- Демонстрирует условное отображение контента

## Работа системы

1. **Первый заход:** Панель показывается, все cookie по умолчанию разрешены
2. **Нажатие "Разрешить все":** Сохраняются настройки, отправляется на сервер, панель скрывается
3. **Изменение настроек:** При изменении чекбоксов автоматически обновляются cookie и отправляется запрос
4. **Логирование:** Все действия пользователя сохраняются в таблицу `tsm_cookie_consent_log`
